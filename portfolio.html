Projet Sélectionné pour l'Oral

Création d'un Site Web Professionnel

Réalisation d'une SAÉ (Situation d'Apprentissage et d'Évaluation) en BUT Informatique

Le Besoin : Concevoir et déployer un site internet fonctionnel répondant strictement à un cahier des charges client, tout en garantissant une navigation fluide et un design responsive.

Voir le Site Code Source

Stack Technique

HTML5 CSS3 JavaScript Trello Git / GitHub

Démarche & Modalités

Dans le cadre de cette SAÉ à l'IUT de Montreuil, nous avons travaillé en équipe avec une répartition agile des tâches via Trello.

  • Analyse : Étude approfondie des besoins du client et rédaction des spécifications.
  • Communication : Échanges continus avec les clients à travers Microsoft Teams et WhatsApp pour valider les étapes (maquettes, wireframes).
  • Développement : Codage propre et structuré sous VS Code en utilisant le triptyque standard (HTML, CSS, JS).

Si c'était à refaire... (Recul Critique)

Ce projet m'a permis de valider des compétences clés du Programme National (PN), notamment sur l'axe "Réaliser un développement d'application" et "Travailler dans une équipe informatique".

"Si je devais recréer ce site aujourd'hui, j'intégrerais une architecture basée sur le pattern MVC ou un framework comme React pour modulariser les composants. De plus, j'optimiserais la gestion de projet en automatisant l'intégration continue (CI/CD) via GitHub Actions."
